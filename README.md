# Stopwatch

A typical stopwatch working properly with front end language.

## Visit the Live Website
🌐 **https://Levu55.github.io/-A-Stopwatch/**

## Features
- Start/Pause button
- Lap time recording
- Reset functionality
- Beautiful UI with gradient background

## How to Use
1. Click **Start** to begin timing
2. Click **Lap** to record lap times
3. Click **Reset** to clear and start over
